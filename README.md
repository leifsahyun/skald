# Skald

A simple static HTML website boilerplate for GitHub Pages.

## Features

- Clean, modern design with responsive layout
- Three pages: Home, About, and Contact
- CSS styling with gradient hero section
- JavaScript for form handling and smooth scrolling
- Mobile-friendly navigation
- Ready for GitHub Pages deployment

## File Structure

```
├── index.html          # Home page
├── about.html          # About page
├── contact.html        # Contact page
├── css/
│   └── style.css       # Stylesheet
├── js/
│   └── script.js       # JavaScript
└── README.md           # This file
```

## Deployment

This site is ready to be deployed to GitHub Pages:

1. Go to your repository settings
2. Navigate to the "Pages" section
3. Select the branch you want to deploy (usually `main` or `master`)
4. Select the root directory as the source
5. Save and your site will be live at `https://[username].github.io/[repository-name]/`

## Customization

- Edit the HTML files to change content
- Modify `css/style.css` to adjust styling
- Update `js/script.js` to add more interactivity
- Replace the gradient colors in the hero section to match your brand
